Detailed Description

A Seaboard Synth object that facilitates polyphonic parameter control. This class is used by SeaboardPlayer objects. In order to produce sound from seaboard input you should create a SeaboardPlayer object and add it as a listener to a Seaboard object. Custom voices can be added to a SeaboardSynth through SeaboardPlayer, by using the SeaboardPlayer::addNewVoice() function. The SeaboardVoice::renderNextBlock() function will be called by the parent class of SeaboardSynth, juce::Synthesiser.
